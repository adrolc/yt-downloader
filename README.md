## Installation
Create a virtual environment
```
python -m venv .venv
```
Activate virtual environment

Windows
```
$ .venv\Scripts\activate
```
Linux
```
source ./.venv/bin/activate
```
Install dependency
```
$ pip install -r requirements.txt
```
